Instructions

   Instructions for using Trinetr app.

1. Login using your email Id and OTP .
2. Click on Add New Mobile device to add Mobile as device.
3. Enter name You want to be displayed for device.
4. Enter email Id of device which you want to track.
5. Click on add button.
6. Click on refresh button to load tracking information.
6. All done.



   Instructions to use Trackify App.

1. CLick on Link below and Install the app.
2. Turn on device location services.
3. Open installed app. and Login with email ID and OTP
4. There is a text field above start button select it.
5. Enter the email Id on which you want to track the device.
6. Click on Start button.
7. Close the app.
8. You can see in notification bar to see if app is running.
9. All done.
